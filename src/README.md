# EmployeeTaskTracker

Система отслеживания задач сотрудников с использованием продвинутых концепций ООП в Kotlin.

## Функции:
- Управление сотрудниками с инкапсуляцией данных
- Создание и отслеживание задач
- Иерархия отделов с абстрактными классами
- Генерация отчётов через интерфейсы

## Технологии:
- Kotlin
- ООП (инкапсуляция, наследование, полиморфизм)
- Git для контроля версий
